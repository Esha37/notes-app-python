# Notes App (File Handling)

## Overview

A console-based Notes Management application developed using Python and File Handling. The application allows users to create, view, update, and delete notes while storing data in a text file.

## Features

- Add Note
- View Notes
- Update Note
- Delete Note
- Store Notes in a Text File

## Technologies Used

- Python
- File Handling

## Skills Demonstrated

- Python Functions
- File Handling
- CRUD Operations
- Loops
- Conditional Statements
- Menu-Driven Programming

## How to Run

```bash
python notes_app.py
```
